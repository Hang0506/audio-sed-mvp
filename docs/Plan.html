🎯 Trạng Thái Component

ComponentStatusChi Tiết
User Intake (Multi-select) Real Form checkbox card phân nhóm Mũi / Họng-Tai. Lưu symptom_ids[] + auto-map disease_tags.
Context Engine (Weather) Real CoreLocation GPS → OpenWeatherMap API → AQI, PM2.5, nhiệt độ, độ ẩm. Trigger khi user có "mui_hat_hoi_lanh".
AI Food Scanner Real YOLOv10 (VietFood67, mAP50=0.92). Camera live viewfinder + cross-reference dị nguyên với symptom_ids.
AI Audio Engine Real — Đã Có YAMNet ONNX + Cough V2 + OSA Screening. Backend FastAPI đã sẵn sàng. iOS ghi âm → POST WAV → nhận kết quả.
Rule Engine Real Map symptom_id → kích hoạt feature tương ứng. If/else hoặc JSON config.
Gamification (Coins) Fake UI Xu cộng local. Chưa có ví thật/giao dịch backend. Đủ để validate hành vi user.
Voucher Marketplace Fake UI Danh sách voucher hardcode. Hành động "đổi" chỉ trừ xu local + hiện thông báo.
Weekly Report Phase 2 Cần tích lũy data 7 ngày. MVP hiện thị data mẫu; Phase 2 lấy data thật từ audio sessions.
